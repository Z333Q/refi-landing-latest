# ReFi.Trading Compliance Copy Mapping

## Qatar QFC Positioning: Non-Regulated B2B SaaS Technology Provider

This document tracks all copywriting changes made to reposition ReFi.Trading as an execution automation software provider (non-advisory, non-discretionary) for Qatar Financial Centre (QFC) regulatory compliance.

---

## Global Term Replacements

| Original Term | New Term | Rationale |
|--------------|----------|-----------|
| "AI trading platform" | "AI execution automation platform" | Positions as tool provider, not trading service |
| "AI trading" | "execution automation" | Removes trading advice implication |
| "trading strategies" | "trading rules" or "trading algorithms" | User-configured, not platform-supplied |
| "strategy-as-a-service" | "execution automation software" | Software licensing, not advisory service |
| "agents decide to trade" | "agents submit orders based on user-configured rules" | User authority emphasized |
| "optimize" | "compute", "verify", "enforce", "execute within constraints" | Removes discretionary management language |

## Security & Compliance Features Emphasized

1. **Non-Custodial Architecture**: Assets remain at user's broker at all times
2. **User Authority**: Users define all rules, parameters, and risk limits
3. **Broker Execution**: Orders execute through user's brokerage account
4. **Proof Gates**: Zero-knowledge proofs block non-compliant orders
5. **Software Provision**: ReFi.Trading provides software, not advisory services
6. **No Discretion**: No investment advice, recommendations, or portfolio management

## Implementation Notes

All changes preserve technical accuracy while repositioning ReFi.Trading as a technology provider rather than a financial service. The emphasis is on:

1. **Tool Language**: Software for execution automation
2. **User Control**: Users define all rules and retain authority
3. **Non-Custodial**: Assets never leave user's broker
4. **Proof Enforcement**: Cryptographic gates block non-compliant orders
5. **No Advisory**: Explicit disclaimers about no investment advice

This positioning allows ReFi.Trading to operate as a B2B SaaS provider in Qatar QFC without triggering financial services regulation.
